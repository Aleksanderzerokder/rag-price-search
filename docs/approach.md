# Подход (кратко)

1) Нормализуем прайс (title/unit/price + keywords/синонимы)
2) Индексация: embeddings каждой позиции → векторное хранилище
3) Поиск: embeddings запроса → top-N похожих
4) Правила:
- top-1 выше порога → выбираем автоматически
- несколько близких → показываем варианты/уточняем
- ниже порога → "не найдено" + уточняющие вопросы
