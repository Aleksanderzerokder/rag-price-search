# Демо: семантический поиск (RAG) по прайсу/базе

Демонстрация подбора позиций из прайса **по смыслу запроса**, а не по совпадению слов.

**запрос → семантический поиск → top-N → правила выбора → результат**

## Что решает
- Поиск по словам “не находит” нужные позиции
- Долгие уточнения и ручной подбор
- Ошибки подбора и расчёта из-за разных формулировок
